O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação de hipertexto. É usado para estruturar páginas na web.

Estrutura Básica

Essa é a estrutura que um documento HTML deve possuir:

<!DOCTYPE html>
<html>
  <head>
    <title>Título</title>
  </head>
  <body>
    <h1>Cabeçalho</h1>
    <p>Parágrafo</p>
  </body>
</html>

HTML Básico

Dentre os elementos que englobam o HTML básico, temos:

  • <p>: parágrafos
  • <a>: links
  • <img>: imagens
  • <ul> e <ol>: listas

Listas em HTML

O HTML possui dois tipos principais de listas:

  1. Listas ordenadas (numeradas)
  2. Listas não ordenadas (com marcadores)

Outros elementos

Texto sublinhado, texto forte e itálico são usados para dar destaque.
Também podemos usar subscrito e sobrescrito.

Atributos HTML

Elementos HTML podem ter atributos que fornecem informações adicionais:

    <a href="https://dio.me" target="_blank">Visite a DIO</a>
    <img src="nome-imagem.jpg" alt="Descrição">
  • href: destino do link
  • src: caminho da imagem
  • alt: descrição da imagem
  • target: como abrir (ex: nova aba)

Tags Semânticas em HTML

Essas tags explicam a função do conteúdo:

  • <header>
  • <nav>
  • <main>
  • <section>
  • <article>
  • <aside>
  • <footer>
